(相似矩阵：矩阵 ( A ) 和矩阵 ( B ) 通过可逆矩阵 ( P ) 相关联)


(矩阵转置：矩阵的转置是将行与列互换)


(矩阵加法：矩阵加法是对应元素相加)


(矩阵乘法：矩阵 ( A ) 和矩阵 ( B ) 的乘积)


(矩阵的逆：矩阵 ( A ) 的逆矩阵存在的条件是行列式不为零)


(行列式：2x2矩阵的行列式计算)


(迹：矩阵的迹是对角线元素之和)


(正交矩阵：矩阵 ( A ) 满足 ( A^T A = I )，即其转置矩阵与其本身的乘积为单位矩阵)


(特征值方程：矩阵 ( A ) 与向量 ( x ) 的关系)


(矩阵的幂：矩阵的多次自乘)


(相似矩阵：矩阵 ( A ) 和矩阵 ( B ) 是相似的，当且仅当存在可逆矩阵 ( P )，使得 ( A = P^{-1} B P ))


(对称矩阵：矩阵 ( A ) 等于其转置矩阵 ( A^T ))


(酉矩阵：矩阵 ( A ) 满足 ( A^T A = A A^T )，即它是正交矩阵的扩展)


(矩阵的秩：矩阵 ( A ) 的秩是其行或列中线性无关的最大数目)


(单位矩阵：矩阵 ( A ) 与单位矩阵 ( I ) 相乘，得到矩阵 ( A ) 本身)


(伴随矩阵：矩阵 ( A ) 的伴随矩阵是其余子式矩阵的转置)


(行列式与特征值：矩阵 ( A ) 的行列式等于其特征值的乘积)


(矩阵的逆：矩阵 ( A ) 的逆等于其伴随矩阵除以行列式)


(乘积 ( A ) 和 ( A^T )：如果矩阵 ( A ) 为实矩阵，则其与转置的乘积是对称矩阵)


(矩阵的特征值：矩阵 ( A ) 的特征值是方程 ( A \cdot x = \lambda \cdot x ) 的解)


(矩阵的秩与奇异值：矩阵 ( A ) 的秩等于其奇异值的非零个数)
